1. In a paragraph (at least 3 sentences), describe the process you took to solving the code.

I read the requirements of the project first, especially for the requirements of the password validation and generation. I also looked at the requirements to get at least a B as I thought that would serve as a good baseline. I stored my username and password first, then did the code for the password validation by using boolean values and using a for loop as well as .charAt in order to check each character if it was an uppercase letter, a lowercase letter, or a number. Using those boolean values after the for loop checks it and updates them, I used if else if statements to print out the appropriate message if the password met the criteria or not. After that, I added in the feature for special symbols by putting the common ones in a string and using another for loop inside the original one to check if each character was present in the specialSymbols string.

Finally, I implemented the password generator. I used a while loop to make sure the prompt is asked for the length in case the user inputs a length less than 8. Another while loop was used as well to ask if the user wanted special symbols in their password. I also decided on this in order to make sure the user only chooses between 0 and 1. I created arrays for the letters, numbers, and used .toCharArray to turn the string of specialSymbols into an array. Another for loop is used to randomize what to use for each index of the string. StringBuilder was used in order to keep appending new elements instead of constant concatenation. Lastly, an if statement was used to guarantee that a special symbol is included if the user approves.
